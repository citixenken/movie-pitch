# Movie Idea Generator with ChatGPT and Dall-E

Quick start:

In `env.js`, paste your OpenAI API Key:
```
export const process = {
  env: {
    OPENAI_API_KEY: <YOUR-OPENAI-API-KEY>,
  },
};
```
Then,

```
$ npm install
$ npm start
```



Head over to https://vitejs.dev/ to learn more about using vite
